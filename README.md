# MobiCarePlus_Backend
MobiCarePlus MobiCarePlus is an enterprise-level web application that acts as a bridge between customers and local mobile repair and accessory shops. It allows shopkeepers to manage their profiles, share pricing and services, and sell second-hand mobiles. Customers can browse nearby shops, search for services, and provide ratings and reviews.  

#Features
 ***Customer Functionality
Browse nearby shops.
Search by services, prices, and shop names.
Provide ratings and reviews for services.
Send queries to shopkeepers via messaging.
View shops on a map.
View second-hand mobiles and accessories.
Add items, accessories, or services to a wishlist.

***Shopkeeper Functionality
Manage profiles, services, and pricing.
Upload images, videos, and accessories.
Offer hot deals and promotions.
View ratings and reviews.
View and reply to customer messages.

****Admin Functionality
Verify shopkeepers.
Add or delete shops and products.

#Technologies Used
***Backend
Spring Boot: Framework for building RESTful APIs.
Spring Data JPA: For database interaction.
MySQL: Relational database for storing data.
Lombok: For reducing boilerplate code.
Spring Web: For handling HTTP requests and responses.

***Frontend
Angular: Framework for building the user interface.
Google Maps API: For displaying shop locations on a map.
Messaging API: For enabling chat functionality.
